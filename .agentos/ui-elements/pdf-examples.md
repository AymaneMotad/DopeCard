# PDF Generation Examples

## Overview

Reference designs for PDFs generated when a card is created. These PDFs typically contain QR codes for distribution.

## Example Files

**Location**: `/Generated PDF once card created exampels/` folder contains:
- `A4-pdf-inspiration.jpeg`
- `card-creation-dashboard-inspiration.avif`
- `dashboard-inspiration.avif`
- `membership-inspiration-pdf.webp`

## PDF Content

### Standard Elements
- Business logo
- Card title/name
- QR code (large, scannable)
- Instructions for customers
- Business contact information
- Terms and conditions (optional)

### Layout Options
- A4 size (printable)
- Portrait orientation
- Centered QR code
- Branded header/footer

## Use Cases

1. **Print for Physical Display**: Post in-store for customers to scan
2. **Digital Distribution**: Share via email, social media
3. **Event Handouts**: Print for events or promotions
4. **Window Display**: Print for store windows

## Technical Implementation

### Generation
- Generate PDF server-side
- Include QR code (high resolution)
- Use business branding
- Optimize file size

### Features
- Download button after card creation
- Print-friendly format
- Multiple size options (A4, Letter, etc.)
- Customizable layout

## Design Guidelines

1. **QR Code Size**: Large enough to scan from 2-3 feet away
2. **Instructions**: Clear, simple instructions
3. **Branding**: Consistent with card design
4. **Readability**: High contrast, clear fonts
5. **Professional**: Clean, professional appearance
